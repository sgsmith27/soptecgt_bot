# Problema: El scanner no escanea documentos de identificación (cédulas, pasaportes) – Fujitsu fi-65F

## Categoría
- scannercaja

## Síntomas
- El scanner **no captura la imagen** del documento colocado en la cama plana.
- El software muestra la vista previa en blanco o negra.
- La luz de escaneo no se mueve o no ilumina el documento.
- El sistema de caja indica “No se detecta imagen” o “Scanner no disponible”.

## Causas probables
- El documento no está bien colocado sobre la superficie de escaneo.
- El scanner no está seleccionado como **dispositivo predeterminado** en el software.
- Conexión USB floja o en un puerto incorrecto.
- El scanner no está recibiendo energía suficiente (puertos USB frontales).
- Software de captura bloqueado o detenido.
- Vidrio sucio (muy frecuente con cédulas laminadas o plastificadas).

## Pasos de solución

1. **Verificar conexión USB**
   - Asegurarse de que el cable USB esté conectado firmemente al scanner y la PC.
   - Preferir puertos USB **traseros** del CPU (más estables).
   - Evitar hubs o extensiones USB.

2. **Colocar correctamente el documento**
   - Colocar la **cédula o pasaporte boca abajo**, con la fotografía hacia el vidrio.
   - Alinear la esquina superior del documento con la **guía marcada** en el fi-65F.
   - Mantener el documento plano sin dejar espacios de aire.

3. **Limpiar el vidrio superficial (sin abrir el equipo)**
   - Apagar el scanner.
   - Usar un paño limpio y seco (sin líquidos).
   - Remover polvo, grasa o manchas que interfieran con la lectura.
   - En pasaportes, presionar ligeramente para evitar sombras.

4. **Reiniciar el scanner**
   - Desconectar el cable USB por 5 segundos.
   - Conectarlo nuevamente.
   - Esperar a que el sistema corporativo lo detecte.

5. **Reiniciar el software de captura**
   - Cerrar la aplicación de escaneo en el sistema de caja.
   - Abrirla nuevamente.
   - Verificar que el **Fujitsu fi-65F** esté seleccionado como dispositivo de entrada.

6. **Probar con un documento estándar**
   - Intentar escanear un papel tamaño A6 o una prueba generada por el sistema.
   - Si el scanner captura este tipo de documento, el problema es de posición o superficie del documento de identificación.

## Cuándo escalar
- El scanner no ilumina ni realiza movimiento interno.
- El software no detecta el dispositivo aunque esté conectado.
- El vidrio presenta rayones o manchas internas que oscurecen la imagen.
- El escaneo aparece totalmente blanco o negro de forma constante.
- Escalar para:
  - Limpieza técnica interna (sensor CIS).
  - Revisión de cable USB o reemplazo por cable original.
  - Diagnóstico de firmware del fi-65F.
  - Reemplazo del scanner en caso de falla de iluminación.

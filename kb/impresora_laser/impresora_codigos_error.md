# Problema: Impresora muestra códigos de error en pantalla (Lexmark MX611 / MS810 / MX622)

## Categoría
- impresora_laser

## Síntomas
- La impresora muestra mensajes como:
  - **200.x** – Atasco de papel
  - **900.xx** – Error del sistema o firmware
  - **202.x** – Atasco en ruta interna
  - **31x.x** – Error de escáner (solo MX611/MX622)
  - **80.x** – Error de fusor
- La impresora deja de responder o no permite imprimir.

## Causas probables
- Error de firmware.
- Fallo de componentes internos.
- Sensores o rodillos defectuosos.
- Problemas con el escáner en equipos multifunción.

## Pasos de solución

1. **Reiniciar la impresora**
   - Apaga la impresora.
   - Desconéctala 10 segundos.
   - Enciéndela nuevamente.

2. **Verificar si el error es temporal**
   - Si el error desaparece después del reinicio, intentar imprimir una página de prueba.

3. **Errores 200.x / 201.x / 202.x**
   - Revisar bandeja de papel.
   - Revisar compuertas.
   - Retirar atascos visibles siguiendo el panel.

4. **Errores 80.x (fusor)**
   - Normalmente requiere intervención técnica.
   - No tocar el fusor: puede estar caliente.

5. **Errores 900.xx**
   - Indican problemas internos o de firmware.
   - Registrar incidente de inmediato.

6. **Errores 31x (escáner en MX611/MX622)**
   - Reiniciar equipo.
   - Verificar que la tapa superior esté cerrada.

## Cuándo escalar
- Si el código persiste después del reinicio.
- Si el panel indica “Service Required”.
- Si el error es **900.xx** o **80.xx**.
- Escalar para:
  - Diagnóstico técnico interno.
  - Reemplazo de fusor, sensores o unidad de escaneo.
  - Actualización de firmware.

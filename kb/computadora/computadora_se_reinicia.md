# Problema: La computadora se reinicia sola o se apaga y enciende de nuevo

## Categoría
- computadora

## Síntomas
- El equipo se reinicia sin intervención del usuario.
- Se apaga y vuelve a encender durante el uso.
- Puede ocurrir durante tareas simples (correo, navegador) o al abrir aplicaciones.
- No aparecen mensajes de error visibles para el usuario.

## Causas probables
- Fallos en energía o conexión eléctrica inestable.
- Adaptador de corriente defectuoso.
- Sobrecalentamiento leve (polvo en disipador o ventilación obstruida).
- Problemas de Windows o controladores.  
  *(Los usuarios no pueden modificar configuraciones avanzadas).*

## Pasos de solución

1. **Revisar la conexión eléctrica**
   - Conecta el adaptador directamente al tomacorriente.
   - Evita regletas saturadas o extensiones dañadas.
   - Si otros equipos en el mismo tomacorriente presentan fallas, prueba otra toma.

2. **Verificar el adaptador de corriente HP**
   - Si el adaptador está muy caliente, desconéctalo y espera 5 minutos.
   - Verifica que el cable no esté doblado o suelto.
   - Si tienes otro adaptador compatible disponible, prueba con él.

3. **Mantener ventilación adecuada**
   - Asegúrate de que el CPU HP ProDesk 600 G5 Mini tenga espacio libre a los lados y atrás.
   - No lo coloques sobre superficies suaves (toallas, papel, etc.) que bloqueen la ventilación.

4. **Reiniciar Windows de forma segura**
   - Presiona `Ctrl + Alt + Supr` → Reiniciar.
   - Espera a que cargue por completo antes de abrir aplicaciones pesadas.

5. **Actualizar Windows y aplicaciones corporativas**
   - Si el equipo pertenece a dominio corporativo, las actualizaciones se aplican automáticamente.
   - Deja el equipo conectado a energía durante la jornada para que complete actualizaciones.

## Cuándo escalar
- Si el reinicio ocurre al abrir una aplicación específica.
- Si el equipo se reinicia incluso estando inactivo.
- Si se reinicia inmediatamente después de encender.
- Escalar para:
  - Revisión del adaptador o fuente interna.
  - Limpieza interna (técnico autorizado).
  - Diagnóstico de hardware (RAM, placa base).

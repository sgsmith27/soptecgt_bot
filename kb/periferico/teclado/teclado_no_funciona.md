# Problema: Teclado USB no funciona (no responde o no escribe)

## Categoría
- periferico

## Subcategoría
- teclado

## Síntomas
- El teclado no escribe en pantalla.
- Las teclas **no responden** o escriben caracteres incorrectos.
- Las luces del teclado (NumLock, CapsLock) no encienden.
- El sistema no muestra ningún mensaje cuando se conecta el teclado.
- El mouse sí funciona, pero el teclado no.

## Causas probables
- Cable USB flojo o conectado en un puerto defectuoso.
- Puerto USB frontal sin energía suficiente.
- Conflicto temporal del controlador USB.
- Daño en el cable del teclado.
- Teclado bloqueado o deshabilitado por error del sistema.
- Polvo o suciedad acumulada (común en sucursales).

## Pasos de solución

1. **Verificar conexión física**
   - Asegúrate de que el teclado esté conectado firmemente al **puerto USB trasero** del CPU.
   - Evitar puertos USB frontales (menos estables en sucursales).

2. **Probar otro puerto USB**
   - Desconecta el teclado y colócalo en otro puerto USB **trasero**.
   - Si encienden las luces del teclado, el puerto anterior estaba fallando.

3. **Probar con otro teclado USB (si hay disponible)**
   - Conecta un teclado de otra estación de trabajo.
   - Si el segundo teclado funciona → el teclado original está dañado.

4. **Reiniciar la computadora**
   - Reiniciar el equipo puede restablecer el controlador USB.
   - Después del reinicio, probar nuevamente el teclado.

5. **Comprobar bloqueo de teclas**
   - Presionar **NumLock** o **CapsLock** y verificar si la luz cambia.
   - Si las luces no encienden, el teclado no está recibiendo energía.

6. **Limpiar superficialmente el teclado**
   - Apagar el equipo.
   - Sacudir suavemente o limpiar con paño seco para remover polvo.
   - No usar líquidos.

7. **Evitar hubs o extensiones USB**
   - Los teclados deben conectarse directamente al CPU, no a extensiones o hubs.

## Cuándo escalar
- El teclado no funciona en **ningún puerto USB**.
- Ningún teclado funciona en el CPU (puertos dañados).
- El teclado funciona pero escribe caracteres erróneos constantemente.
- El teclado funciona pero deja de responder frecuentemente.
- Escalar para:
  - Reemplazo del teclado.
  - Revisión de puertos USB del CPU.
  - Diagnóstico de controladora USB.

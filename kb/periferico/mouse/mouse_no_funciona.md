# Problema: Mouse USB no funciona (sin movimiento o clics)

## Categoría
- periferico

## Subcategoría
- mouse

## Síntomas
- El puntero **no se mueve** en pantalla.
- Los clics no funcionan o funcionan de forma irregular.
- El mouse no enciende la luz LED inferior (según modelo).
- Windows no muestra ningún mensaje al conectarlo.
- El teclado sí funciona, pero el mouse no.

## Causas probables
- Puerto USB defectuoso.
- Cable o conector del mouse dañado.
- Mouse obstruido por suciedad en el sensor.
- Controlador USB temporalmente bloqueado.
- Problema del sistema (PC congelada o aplicación trabada).

## Pasos de solución

1. **Verificar que la PC no esté congelada**
   - Presiona **Ctrl + Alt + Supr**.
   - Si aparece el menú, la PC está respondiendo y el problema es del mouse.
   - Si no responde, reiniciar el equipo.

2. **Probar otro puerto USB**
   - Conectar el mouse en un **puerto USB trasero** del CPU.
   - Evitar puertos frontales y extensiones USB.

3. **Revisar que el mouse encienda**
   - Si el modelo tiene luz LED inferior, verificar que se ilumine al moverlo.
   - Si no enciende → problema de alimentación o cable.

4. **Limpiar el sensor**
   - Apagar el equipo.
   - Con un paño seco, limpiar la zona del sensor óptico.
   - Revisar que no haya polvo acumulado (muy común en sucursales).

5. **Probar con otro mouse USB**
   - Conectar un mouse de otra estación.
   - Si el segundo funciona → el mouse original está dañado.

6. **Reiniciar la PC**
   - Reiniciar puede restablecer los controladores USB.

7. **Evitar hubs o alargadores USB**
   - El mouse debe ir **directo al CPU**.
   - Evitar cajas registradoras o teclados que actúan como intermediarios.

## Cuándo escalar
- El mouse no funciona en **ningún puerto USB**.
- Ningún mouse funciona en la computadora (posible daño de puertos).
- El puntero se mueve solo o de manera errática sin tocar el mouse.
- El mouse funciona intermitentemente.
- Escalar para:
  - Revisión de controladora USB.
  - Diagnóstico de puertos dañados.
  - Reemplazo del mouse.

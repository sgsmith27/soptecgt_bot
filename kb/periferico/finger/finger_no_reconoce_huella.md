# Problema: El lector de huellas no reconoce la huella (Digital Persona)

## Categoría
- periferico

## Subcategoría
- finger

## Síntomas
- El lector de huellas **no detecta** el dedo al colocarlo.
- La luz LED enciende, pero la aplicación dice “No se reconoció la huella”.
- La huella se escanea pero **no es aceptada** por el sistema.
- La huella tarda demasiado en capturarse o aparece borrosa.
- El sistema muestra “Sensor no responde” o “Capture failed”.

## Causas probables
- Vidrio del lector sucio (muy común en tiendas).
- Dedo húmedo, muy seco o con suciedad.
- El lector está siendo bloqueado por otra aplicación.
- El puerto USB no está entregando suficiente energía.
- Interferencia de guantes, curitas o protectores.
- Problema temporal de comunicación entre el lector y la app corporativa.

## Pasos de solución

1. **Limpiar el lector de huellas**
   - Usar un paño **suave y seco**.
   - Limpiar el vidrio donde se coloca el dedo.
   - No usar alcohol, gel ni líquidos.

2. **Limpiar el dedo del usuario**
   - Asegurarse de que el dedo esté seco.
   - Evitar colocar demasiada presión.
   - Probar con otro dedo si el primero no funciona.

3. **Reiniciar la aplicación corporativa**
   - Cerrar la app que usa el lector de huellas.
   - Abrirla de nuevo e intentar capturar la huella.

4. **Cambiar de puerto USB**
   - Conectar el lector en un **puerto USB trasero** del CPU.
   - Evitar puertos frontales y extensiones USB.
   - Verificar que el LED del lector encienda correctamente.

5. **Reiniciar el dispositivo**
   - Desconectar el lector por 5 segundos.
   - Volver a conectarlo.
   - Esperar a que el sistema lo detecte automáticamente.

6. **Probar en otra PC (si disponible)**
   - Si funciona en otro equipo → problema en la PC original.

## Cuándo escalar
- El lector captura la imagen pero **siempre rechaza la huella**.
- La aplicación muestra repetidamente “Sensor no responde”.
- El lector se desconecta con frecuencia.
- El cristal está rayado o visiblemente dañado.
- Escalar para:
  - Diagnóstico de software.
  - Reemplazo del lector Digital Persona.
  - Verificación de controladora USB.

# Problema: Windows no arranca (pantalla negra, reinicio continuo o error de inicio)

## Categoría
- software

## Síntomas
- Al encender la computadora, Windows **no carga**.
- Aparece una pantalla negra sin avanzar.
- El equipo se **reinicia continuamente** antes de llegar al escritorio.
- Aparecen mensajes como:
  - “Preparing Automatic Repair”
  - “Diagnosing your PC”
  - “Your PC ran into a problem”
  - “No bootable device”
- El logo de Windows aparece pero luego se congela.

## Causas probables
- Apagado incorrecto o corte de energía.
- Archivos de inicio dañados.
- Actualización de Windows incompleta.
- Falla del disco o del sistema de archivos.
- Configuración incorrecta en el arranque (no modificada por el usuario).
- Error del sistema por antivirus o actualizaciones corporativas.

## Pasos de solución

1. **Reiniciar completamente el equipo**
   - Mantén presionado el botón de encendido por **5–10 segundos** hasta que el equipo se apague.
   - Espera 5 segundos y vuelve a encenderlo.
   - Muchos errores temporales se corrigen con un reinicio completo.

2. **Verificar la pantalla**
   - Asegúrate de que el monitor esté encendido y en la **entrada correcta** (HDMI/DP).
   - Mover el mouse o presionar `Enter` para descartar modo suspensión.
   - Si el monitor dice “Sin señal”, revisar cables (ver categoría *monitor*).

3. **Esperar si Windows muestra reparación automática**
   - Si aparece “Preparing Automatic Repair” o “Diagnosing your PC”, **esperar al menos 5–10 minutos**.
   - Es normal que tarde si hubo apagados inesperados.

4. **Descartar periféricos que bloquean el arranque**
   - Desconectar memorias USB, discos externos o lectores de tarjetas.
   - Reiniciar nuevamente.

5. **Intentar entrar al menú de recuperación**
   - Si Windows intenta iniciar pero falla repetidamente, **deja que cargue la reparación automática**.
   - Si aparece el menú “Recovery”, informar inmediatamente a soporte (el usuario no debe hacer cambios ahí).

6. **Revisar si el equipo muestra mensajes tipo “No boot device”**
   - Esto puede indicar falla del disco o del sistema de arranque.
   - Registrar incidente directamente (no es reparable por usuario).

## Cuándo escalar
- Windows entra en bucle de reparación automática de forma repetida.
- El equipo muestra “No bootable device” o “Boot device not found”.
- Pantalla negra persistente incluso con el monitor correcto y reinicios.
- Reinicio continuo sin llegar al escritorio.
- Olor a quemado o ruidos inusuales del disco.
- Escalar para:
  - Diagnóstico del disco (HDD/SSD).
  - Reparación del inicio (Boot Manager).
  - Reinstalación del sistema bajo protocolo corporativo.
  - Sustitución del disco si se confirma falla.
